## installation


Clone repository to config nvim
```bash
git clone https://github.com/Rauliqbal/neovim-config.git ~/.config/nvim/
```

Remove .git folder 
```bash
rm -rf ~/.config/nvim/.git
```

Start neovim
```bash
nvim
```
